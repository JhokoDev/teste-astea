# Astea Scientific

Plataforma avançada para gestão de feiras científicas e acadêmicas, com foco em inovação, biotecnologia e excelência acadêmica.

## 🚀 Sobre o Projeto

O **Astea Scientific** é uma solução E2E (End-to-End) projetada para simplificar e profissionalizar a organização de eventos científicos. Desde a concepção da feira até a divulgação dos resultados, a plataforma oferece ferramentas robustas para organizadores, estudantes, orientadores e avaliadores.

## 🛠️ Funcionalidades Principais

### 1. Gestão Centralizada de Feiras
*   **Criação Guiada**: Fluxo passo a passo para configuração de identidade, datas críticas e estrutura do evento.
*   **Formulários Personalizados**: Definição de campos específicos para atender às necessidades de diferentes áreas do conhecimento.
*   **Regras de Avaliação**: Configuração de avaliação cega (blind review), pesos por critério e hierarquia de desempate.

### 2. Submissão e Ciclo de Vida de Projetos
*   **Portal do Estudante**: Interface intuitiva para submissão de resumos, membros da equipe e orientadores.
*   **Gestão de Evidências**: Suporte para upload de documentos, links externos e materiais complementares.
*   **Versionamento**: Histórico completo de alterações em cada projeto submetido.

### 3. Ecossistema de Avaliação
*   **Candidatura de Avaliadores**: Processo de inscrição e curadoria de especialistas para cada feira.
*   **Interface de Avaliação**: Ferramenta dedicada para notas por critérios, rubricas detalhadas e feedbacks construtivos.
*   **Distribuição Inteligente**: Gestão de atribuição de projetos para garantir a imparcialidade e cobertura necessária.

### 4. Painel de Controle e Analytics
*   **KPIs em Tempo Real**: Monitoramento de inscritos, projetos submetidos e progresso das avaliações.
*   **Alertas Críticos**: Sistema de notificações para prazos próximos e pendências administrativas.
*   **Logs de Auditoria**: Rastreabilidade completa de todas as ações realizadas no sistema para garantir transparência e segurança.

### 5. Experiência e Acessibilidade
*   **Interface Adaptativa**: Design responsivo otimizado para desktops, tablets e dispositivos móveis.
*   **Modo Escuro (Dark Mode)**: Suporte nativo a temas claros e escuros para maior conforto visual.
*   **Exploração Pública**: Vitrine de feiras ativas para engajamento da comunidade acadêmica.

## 👥 Perfis de Acesso (RBAC)

A plataforma utiliza um sistema rigoroso de controle de acesso baseado em funções:

*   **Administrador**: Gestão global do sistema e configurações de alto nível.
*   **Gestor (Manager)**: Responsável pela organização e operação das feiras de sua instituição.
*   **Avaliador (Evaluator)**: Especialista focado na análise técnica e pedagógica dos projetos.
*   **Orientador (Advisor)**: Mentor que acompanha e valida a participação dos estudantes.
*   **Estudante (Student)**: Protagonista do evento, responsável pela submissão e apresentação dos trabalhos.

*Este projeto é um serviço proprietário. O acesso ao código-fonte é restrito, sendo disponibilizado apenas como serviço (SaaS).*
